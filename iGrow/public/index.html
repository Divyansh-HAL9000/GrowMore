Real-time Monitoring

Monitors the real-time values of the environmental factors within the enclosure that updates every 5 seconds and pulls the data directly from the Raspberry Pi, apart from that it posts the image of the plant every hour which is clicked using a Pi Camand performs the growth detection algorithm to log the growth of the plant using graphs. The system is designed to trigger a Disease detection AI using image recognition to find the infected leaves and recognize the cause of the disease like Bacterial, viral, or another type of infection.
